GPIO初始化配置CR寄存器，有如下几种状态：

| CFG | MODE | 工作方式                   | 名称 |
| --- | ---- | -------------------------- | ---- |
| 00  | 00   | </font color=r>模拟输入</font> |      |
| 00  | 01   | 通用推挽输出               |      |
| 00  | 10   | 保留                       |      |
| 00  | 11   | 保留                       |      |
| 01  | 00   | 浮空输入                   |      |
| 01  | 01   | 通用开漏输出               |      |
| 01  | 10   | 保留                       |      |
| 01  | 11   | 保留                       |      |
| 10  | 00   | 上下拉输入                 |      |
| 10  | 01   | 复用推挽输出               |      |
| 10  | 10   | 保留                       |      |
| 10  | 11   | 保留                       |      |
| 11  | 00   | 保留                       |      |
| 11  | 01   | 复用开漏输出               |      |
| 11  | 10   | 保留                       |      |
| 11  | 11   | 保留                       |      |
